楽天アフィリエイトプラグイン .

== Description ==

WordPressで投稿する記事を解析してキーワードを抽出、貴方の楽天アフィリエイトIDを含むテキストリンクに自動変換します。

記事内容の要約となるキーワードを元にネイティブ広告へと繋がるリンクが自動的に生成されます。

アフィリエイトでありながらアフィリエイトにおける作業面を徹底排除。

広告を意識しない、意識させない環境においてピュアなコンテンツ作成に特化、コンテンツ内容と密接に関わる広告へと誘導することでコンバージョンアップを促します。
